On the Subject of the Eyedropper

This holdable is intended to aid those who have a hard time telling colors apart. If shown, the color code will display the color of the pixel your cursor is pointing at.

Press on your keyboard to toggle whether the color code is shown in the bottom left.

Pressing or to change how the information is displayed.
By default, there will only be one hex digit per color channel (R, G, B).
If you prefer, two hex digits per channel can be shown by pressing either button.

Alternatively, Microsoft PowerToys has a very useful Color Picker utility for those running Windows.
